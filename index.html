About Dispatch

Dispatch is a SnowRunner vehicle reference tool.

Default vehicle data is based on community-provided CSV data created and maintained by Vlad Vulcan's Ultimate Snowrunner Data Spreadsheet.

Dispatch is designed to make that data easier to browse, search, and explore in a cleaner interface.

You can also upload your own updated CSV files at any time to override the default data stored in the app.
